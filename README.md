# Larval sleep quantification code

This repository contains custom code used to quantify larval sleep in Drosophila.

## Related manuscript
Neuropeptidergic circuit modulation of developmental sleep in Drosophila

## Contents
- `videoanalyzer_single_loop_ROI_F_binary-thresh_AppliedPC_8.py`  
  Calculates frame-by-frame pixel changes for each well from larval behavior videos and outputs the results as CSV files.
- `20211201_data_processing.R`  
  Reads the CSV files generated by the Python script and calculates sleep amount.

## Environment
- Python Programming Language v3.10.0
- R Project for Statistical Computing v4.1.2
- Required Python and R packages are indicated in the scripts.

## Usage
1. Run `batch_24well_binary-thresh_AppliedPC_8.py` to generate CSV files containing frame-by-frame pixel change values for each well.
2. Run `20211201_data_processing.R` to read the CSV files and calculate sleep amount.

## Input and output
- Input: video files of larval behavior acquired using a 24-well plate system
- Note: well size compatibility should be confirmed before use.
- Intermediate output: CSV files containing frame-by-frame pixel change values for each well

- Final output: quantified sleep measurements
